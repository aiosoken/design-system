### 使用ルール

- **クリアスペース**: ロゴの高さの50%以上の余白を確保
- **最小サイズ**: デジタル 24px以上、印刷 8mm以上
- **禁止事項**: 変形、色変更、エフェクト追加、要素の分離

## カラートークン (CSS Variables)

### ライトモード (デフォルト)

```css
:root {
  /* Primary - AIO Orange */
  --primary: oklch(0.66 0.18 35);
  --primary-foreground: oklch(1 0 0);

  /* Secondary - AIO Blue */
  --secondary: oklch(0.48 0.14 265);
  --secondary-foreground: oklch(1 0 0);

  /* Accent - AIO Orange */
  --accent: oklch(0.66 0.18 35);
  --accent-foreground: oklch(1 0 0);

  /* Background & Foreground */
  --background: oklch(1 0 0);
  --foreground: oklch(0.235 0.015 65);

  /* Card */
  --card: oklch(1 0 0);
  --card-foreground: oklch(0.235 0.015 65);

  /* Popover */
  --popover: oklch(1 0 0);
  --popover-foreground: oklch(0.235 0.015 65);

  /* Muted */
  --muted: oklch(0.967 0.001 286.375);
  --muted-foreground: oklch(0.552 0.016 285.938);

  /* Destructive */
  --destructive: oklch(0.577 0.245 27.325);
  --destructive-foreground: oklch(0.985 0 0);

  /* Border & Input */
  --border: oklch(0.92 0.004 286.32);
  --input: oklch(0.92 0.004 286.32);
  --ring: oklch(0.623 0.214 259.815);

  /* Sidebar */
  --sidebar: oklch(0.985 0 0);
  --sidebar-foreground: oklch(0.235 0.015 65);
  --sidebar-primary: oklch(0.48 0.14 265);
  --sidebar-primary-foreground: oklch(1 0 0);
  --sidebar-accent: oklch(0.967 0.001 286.375);
  --sidebar-accent-foreground: oklch(0.141 0.005 285.823);
  --sidebar-border: oklch(0.92 0.004 286.32);
  --sidebar-ring: oklch(0.623 0.214 259.815);

  /* Chart Colors */
  --chart-1: var(--color-blue-300);
  --chart-2: var(--color-blue-500);
  --chart-3: var(--color-blue-600);
  --chart-4: var(--color-blue-700);
  --chart-5: var(--color-blue-800);
}
```

## ボーダー半径

```css
:root {
  --radius: 0.65rem;          /* 10.4px - 基準値 */
  --radius-sm: calc(var(--radius) - 4px);   /* 6.4px */
  --radius-md: calc(var(--radius) - 2px);   /* 8.4px */
  --radius-lg: var(--radius);               /* 10.4px */
  --radius-xl: calc(var(--radius) + 4px);   /* 14.4px */
}
```

## ボタンスタイル

```css
/* Base button */
.button {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  gap: 0.5rem;
  white-space: nowrap;
  border-radius: var(--radius-md);
  font-size: 14px;
  font-weight: 500;
  transition: all 150ms;
  outline: none;
}

/* Sizes */
.button-default { height: 36px; padding: 8px 16px; }
.button-sm { height: 32px; padding: 6px 12px; }
.button-lg { height: 40px; padding: 8px 24px; }
.button-icon { width: 36px; height: 36px; padding: 0; }

/* Variants */
.button-primary {
  background: var(--primary);
  color: var(--primary-foreground);
}
.button-primary:hover { opacity: 0.9; }

.button-secondary {
  background: var(--secondary);
  color: var(--secondary-foreground);
}
.button-secondary:hover { opacity: 0.8; }

.button-outline {
  background: transparent;
  border: 1px solid var(--border);
}
.button-outline:hover { background: var(--accent); }

.button-ghost {
  background: transparent;
}
.button-ghost:hover { background: var(--accent); }

.button-destructive {
  background: var(--destructive);
  color: white;
}
.button-destructive:hover { opacity: 0.9; }

.button-link {
  background: transparent;
  color: var(--primary);
  text-decoration-line: underline;
  text-underline-offset: 4px;
}
```

## インプットスタイル

```css
.input {
  height: 36px;
  width: 100%;
  border-radius: var(--radius-md);
  border: 1px solid var(--input);
  background: transparent;
  padding: 4px 12px;
  font-size: 14px;
  transition: all 150ms;
}

.input:focus {
  border-color: var(--ring);
  box-shadow: 0 0 0 3px oklch(from var(--ring) l c h / 50%);
}

.input::placeholder {
  color: var(--muted-foreground);
}

.input:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}

/* Dark mode */
.dark .input {
  background: oklch(1 0 0 / 3%);
}
```
